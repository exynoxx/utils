xmlUI
Interactive XML explorer — click elements to build your extraction query
How to use: Load or paste an XML file on the left and click Parse XML. The middle panel shows records paginated in groups of 50 — click any element or attribute to add it as a column. Use ctrl+click to select multiple. A yellow dot on a record means one or more of your selected fields is absent in that record — set a default value in the right panel. Use Select all to auto-add all detected fields, then click ▶ JSON or ▶ XML to download.
